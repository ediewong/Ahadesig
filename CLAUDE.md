# Claude Code 项目指引

本仓库为 **Next.js + Tailwind** 前端模板。实现功能前请先阅读：

1. **`.claude/PROJECT_CONVENTIONS.md`** — 目录结构、设计 Token、代码约定。
2. **`.claude/prompts/`** — 可复制到对话中的任务提示模板（新页面、新组件、feature 迭代）。
3. **`AGENTS.md`** — Next.js 版本说明与官方文档位置。

新增路由、组件或修改主题时，保持 `src/components/design-system/theme.css` 与 `tokens.ts` 同步。
